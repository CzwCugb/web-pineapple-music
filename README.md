# 音乐专辑购物网站：菠萝音乐（Pineapple Music）

## 实现技术

HTML5 + CSS3 + JavaScript


## 开发平台

WebStorm 2024.1.2



## 整体框架

### 代码部分

index ->主页	[导航栏、轮播图、分类导航、页脚]

news -> 新闻页

detail -> 商品详情页

categories -> 商品分类页

cart -> 购物车

profile -> 用户页	[登录，注册，个人信息]



### 注意

1.图片资源放置于images文件夹，需和html文件在同一路径下运行

2.以index.html为入口进入网站
